# Flip image

## Requirements

* [Microsoft Visual Studio](https://https://www.visualstudio.com/)
* [OpenCV](https://opencv.org/)
* [Qt](https://www.qt.io/)

## Installation

* Download the project.
* Open it with `Qt`.
* In the file with `.pro` extension set `INCLUDEPATH` and `LIBS`.
* Build the project and then run it.
